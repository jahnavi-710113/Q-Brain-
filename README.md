# Ultimate Quiz App

A modern and responsive Quiz Application built using HTML, CSS and JavaScript.

## Features
- Random Quiz Questions
- Categories:
  - General Knowledge
  - Cricket
  - History
  - Politics
- Score Tracking
- Personalized Username
- Responsive UI
- Modern Glassmorphism Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
